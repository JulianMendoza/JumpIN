UML Changes to Milestone 3:

Updated Classes:

Board.java - Added board history and clone

MoveSet.java - Added inverted moveset 

MainMenu.java - Added multiple different menus in one

ComponentFactory.java - Added 3D button to factory


New Classes:

BoardHistory.java - Keeps track of moves on board for redo/undo functionality

MoveState.java - Gets board information to be used by solver

Solver.java - Class to find a potential solution to the board using a recursive algorithm

MainMenuListener.java - Listener for main menu events

SolverTests.java - JUnit test for solver

RedoUndoTest.java - JUnit test for redo/undo functionality

Button3D.java - 3D button styling

ButtonFont.java - Font for the new buttons

MenuFrame.java - Abstract menu frame

MenuEvent.java - Events for menu

MenuListener.java - Listener for actions on menu

MenuFont.java - Main Menu font

GameStateMenu.java - Menu inside the main menu that displays the game state

HistoryMenu.java - Menu inside the main menu that holds the redo/undo buttons 

SolverMenu.java - Menu inside the main menu that holds the solver buttons

ThresholdPrompt.java - Prompts user to enter threshold amount for the solver

Styler.java - Styles the main menu

GroupBox.java - Wrapper for the boxes in the main menu

Waiter.java - Wait cursor while solver is loading
